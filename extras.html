¿Qué es un archivo .html?

Un archivo .html es un documento de texto que contiene código HTML, el lenguaje que usan los navegadores para mostrar páginas web.

Visual Studio Code es un editor de código que facilita escribir y organizar este tipo de archivos, con colores y ayudas automáticas.

Atajo útil: En VS Code, escribe ! y presiona Enter para crear la estructura base HTML automáticamente.

Estructura básica de un documento HTML

  • <html>: Raíz del documento.
  • <head>: Información no visible para el usuario, como título y configuraciones.
  • <title>: Texto que aparece en la pestaña del navegador.
  • <body>: Contenido visible en la página.

Esta estructura es la base mínima para que cualquier navegador pueda interpretar y mostrar tu página.

Encabezados en HTML

HTML ofrece seis niveles de encabezados, del <h1> (más importante) al <h6> (menos importante). Sirven para organizar el contenido jerárquicamente.

Diferencia entre <p> y <br>

La etiqueta <p> define un párrafo completo, con margen arriba y abajo.

La etiqueta <br> solo inserta un salto de línea sin crear un nuevo párrafo.

Ejemplos:

        <p>Hola<br>mundo</p>

Resultado: "Hola" y "mundo" en líneas separadas pero dentro del mismo párrafo.

        <p>Hola mundo</p>

Resultado: Texto continuo en un solo párrafo.

Etiqueta <img> y sus atributos

La etiqueta <img> inserta imágenes. Sus atributos principales son:

  • src: Ruta o URL de la imagen.
  • alt: Texto alternativo que se muestra si la imagen no carga.
  • width y height: Ancho y alto de la imagen en píxeles.

Ejemplo:

        <img src="foto.png" alt="Descripción de la imagen" width="300">

Si la imagen no carga, el navegador muestra el texto del atributo alt.

Nota: Es buena práctica siempre incluir el atributo alt para accesibilidad.

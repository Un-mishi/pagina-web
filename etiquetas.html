Concepto de las etiquetas

Las etiquetas HTML son bloques que usamos para estructurar y dar sentido al contenido de una página web. Algunas etiquetas son semánticas (como section o nav) y otras no lo son (como div). Estas etiquetas también nos permiten aplicar estilos más fácilmente con CSS.

Etiquetas básicas

  • h1 a h6: Encabezados
  • p: Párrafo
  • img: Imagen
  • u: Subrayado
  • a: Hipervínculo
  • ul / ol: Listas
  • li: Elementos de lista
  • div: Contenedor genérico
  • span: Contenedor en línea
  • strong: Texto en negrita
  • em, i: Texto en cursiva

Más etiquetas básicas

  • hr: Línea horizontal
  • b: Negrita
  • mark: Resaltar texto
  • blockquote: Cita en bloque
  • code: Texto con formato de código
  • pre: Texto preformateado
  • abbr: Abreviatura
  • audio y video: Reproducción multimedia
  • source: Fuente multimedia
  • track: Subtítulos multimedia
  • figure: Agrupar imágenes con leyenda

Etiquetas para formularios y tablas

  • nav: Barra de navegación
  • table, thead, tbody, tfoot: Tabla
  • tr, td, th: Filas y celdas
  • caption: Título de la tabla
  • form: Formularios
  • input: Campo de entrada
  • label: Etiqueta del campo
  • textarea: Área de texto
  • button: Botón
  • select: Lista desplegable
